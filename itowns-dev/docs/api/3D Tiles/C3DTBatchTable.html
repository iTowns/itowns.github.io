C3DTBatchTable

A 3D Tiles batch table.

Constructor

C3DTBatchTable ( buffer: ArrayBuffer, jsonLength: number, binaryLength: number, batchLength: number, registeredExtensions: Object )

Parameters

  • buffer —

    batch table buffer to parse
  • jsonLength —

    batch table json part length
  • binaryLength —

    batch table binary part length
  • batchLength —

    the length of the batch.
  • registeredExtensions —

    extensions registered to the layer

Properties

type : C3DTilesTypes

Used by 3D Tiles extensions (e.g. C3DTBatchTableHierarchyExtension) to know in which context (i.e. for which 3D Tiles class) the parsing of the extension should be done.

batchLength : number

the length of the batch.

content : object

the content of the batch table in the form: {property1: values[], property2: values[], ...}.

extensions : object

3D Tiles extensions of the batch table stored in the following format: {extensioName1: extensionObject1, extensioName2: extensionObject2, ...}

Methods

getInfoById ( batchID: integer ) : Object

Creates and returns a javascript object holding the displayable information from the batch table and from extensions of the batch table, for a given feature (identified with its batchID).

Returns

  • displayable information relative to the batch table and its extensions. Object is formatted as follow: {batchTable: {property1: value1 property2: value2 ...} extensions: {extension1: {property1: value1 ...} extension2: {...} ...} }

Parameters

  • batchID —

    id of the feature

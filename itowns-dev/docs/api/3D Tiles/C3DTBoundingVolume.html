C3DTBoundingVolume

3D Tiles bounding volume

Constructor

C3DTBoundingVolume ( )

Properties

type : C3DTilesTypes

Used by 3D Tiles extensions (e.g. C3DTBatchTableHierarchyExtension) to know in which context (i.e. for which 3D Tiles class) the parsing of the extension should be done.

box : THREE.Box3

Bounding box, defined only if the Bounding Volume is a box.

region : OBB

Bounding region, defined only if the Bounding Volume is a region.

sphere : THREE.Sphere

Bounding sphere, defined only if the Bounding Volume is a sphere.

extensions : object

3D Tiles extensions of the bounding volume stored in the following format: {extensioName1: extensionObject1, extensioName2: extensionObject2, ...}

TIFFParser

The TIFFParser module provides a parse method that takes a TIFF in and gives a THREE.DataTexture that can be displayed in the view.

It needs the utif library to parse the TIFF.

Example

Fetcher.arrayBuffer('image.tiff')
    .then(TIFFParser.parse)
    .then(function _(texture) {
        var source = new itowns.FileSource({ features: texture });
        var layer = new itowns.ColorLayer('tiff', { source });
        view.addLayer(layer);
    });

Static Methods

parse ( data: ArrayBuffer ) : Promise

Parse a TIFF file and return a THREE.DataTexture.

Returns

A promise resolving with a THREE.DataTexture.

Parameters

  • data —

    The TIFF file content to parse.

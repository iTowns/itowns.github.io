TileMesh

A TileMesh is a THREE.Mesh with a geometricError and an OBB The objectId property of the material is the with the id of the TileMesh

Constructor

TileMesh ( geometry: TileGeometry, material: THREE.Material, layer: Layer, extent: Extent, level: number )

Parameters

  • geometry —

    the tile geometry
  • material —

    a THREE.Material compatible with THREE.Mesh
  • layer —

    the layer the tile is added to
  • extent —

    the tile extent
  • level —

    the tile level (default = 0)

Methods

findCommonAncestor ( tile: TileMesh ) : TileMesh

Search for a common ancestor between this tile and another one. It goes through parents on each side until one is found.

Returns

the resulting common ancestor

Parameters

  • tile

setBBoxZ ( elevation: Object )

If specified, update the min and max elevation of the OBB and updates accordingly the bounding sphere and the geometric error

Parameters

  • elevation
    • elevation.min (optional)
    • elevation.max (optional)
    • elevation.scale (optional)

FeatureBuildingOptions

Constructor

FeatureBuildingOptions ( )

Properties

crs : string

The CRS to convert the input coordinates to.

structure : string

data structure type : 2d or 3d. If the structure is 3d, the feature have 3 dimensions by vertices positions and a normal for each vertices.

filteringExtent : Extent|boolean

Optional filter to reject features outside of extent. Extent filtering is file extent if filteringExtent is true.

buildExtent : boolean

If true the geometry will have an extent property containing the area covered by the geometry. Default value is false if structure parameter is set to '3d', and true otherwise. True if the layer does not inherit from GeometryLayer.

forcedExtentCrs : string

force feature extent crs if buildExtent is true.

filter : function

Filter function to remove features

mergeFeatures : boolean

If true all geometries are merged by type and multi-type.

style : Style

The style to inherit when creating style for all new features.

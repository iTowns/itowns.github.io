OBB extends THREE.Object3D

Oriented bounding box

Constructor

OBB ( min: THREE.Vector3, max: THREE.Vector3 )

Parameters

  • min —

    representing the lower (x, y, z) boundary of the box. Default is ( + Infinity, + Infinity, + Infinity ).
  • max —

    representing the lower upper (x, y, z) boundary of the box. Default is ( - Infinity, - Infinity, - Infinity ).

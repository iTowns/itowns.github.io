CRS

This module provides basic methods to manipulate a CRS (as a string).

Static Methods

defs ( code: string, proj4def: string ) : undefined

Define a proj4 projection as a string and reference.

Parameters

  • code —

    code is the projection's SRS code (only used internally by the Proj4js library)
  • proj4def —

    is the Proj4 definition string for the projection to use

isGeographic ( crs: string ) : boolean

Assert that the CRS is geographic.

Parameters

  • crs —

    The CRS to validate.

isMetricUnit ( crs: string ) : boolean

Assert that the CRS is using metric units.

Parameters

  • crs —

    The CRS to validate.

isValid ( crs: string )

Assert that the CRS is valid one.

Parameters

  • crs —

    The CRS to validate.

reasonnableEpsilon ( crs: string ) : number

Give a reasonnable epsilon to use with this CRS.

Returns

0.01 if the CRS is EPSG:4326, 0.001 otherwise.

Parameters

  • crs —

    The CRS to use.

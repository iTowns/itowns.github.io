Feature2Mesh

Static Methods

convert ( options: Object ) : function

Return a function that converts Features to Meshes. Feature collection will be converted to a a THREE.Group.

Parameters

  • options —

    options controlling the conversion
    • options.batchId (optional) —

      optional function to create batchId attribute. It is passed the feature property and the feature index. As the batchId is using an unsigned int structure on 32 bits, the batchId could be between 0 and 4,294,967,295.

Example

Example usage of batchId with featureId. 
view.addLayer({
    id: 'WFS Buildings',
    type: 'geometry',
    update: itowns.FeatureProcessing.update,
    convert: itowns.Feature2Mesh.convert({
        batchId: (property, featureId) => featureId,
    }),
    filter: acceptFeature,
    source,
});
Example usage of batchId with property. 
view.addLayer({
    id: 'WFS Buildings',
    type: 'geometry',
    update: itowns.FeatureProcessing.update,
    convert: itowns.Feature2Mesh.convert({
        batchId: (property, featureId) => property.house ? 10 : featureId,
        }),
    filter: acceptFeature,
    source,
});

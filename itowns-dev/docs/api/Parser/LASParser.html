LASParser

The LASParser module provides a parse method that takes a LAS file or a LAZ (LASZip) file in, and gives a THREE.BufferGeometry containing all the necessary attributes to be displayed in iTowns. It uses the LASLoader from loaders.gl.

Static Methods

parse ( data: ArrayBuffer, options: Object ) : Promise

Parses a LAS file or a LAZ (LASZip) file and return the corresponding THREE.BufferGeometry.

Returns

A promise resolving with a THREE.BufferGeometry. The header of the file is contained in userData.

Parameters

  • data —

    The file content to parse.
  • options (optional) —

    Options to give to the parser.
    • options.in.colorDepth (default: 'auto') —

      Does the color encoding is known ? Is it 8 or 16 bits ? By default it is to 'auto', but it will be more performant if a specific value is set.
    • options.out.skip (default: 1) —

      Read one point from every skip points.

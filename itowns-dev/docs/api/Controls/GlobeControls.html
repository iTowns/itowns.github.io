GlobeControls

GlobeControls is a camera controller

Constructor

GlobeControls ( view: GlobeView, placement: CameraTransformOptions|Extent, options: object )

Parameters

  • view —

    the view where the control will be used
  • placement —

    the CameraTransformOptions to apply to view's camera or the extent it must display at initialisation, see CameraTransformOptions in CameraUtils.
  • options (optional) —

    An object with one or more configuration properties. Any property of GlobeControls can be passed in this object.

Properties

zoomFactor : number

The factor the scale is multiplied by when dollying (zooming) in or divided by when dollying out. Default is 2.

rotateSpeed : number

Speed camera rotation in orbit and panoramic mode. Default is 0.25.

minDistance : number

Minimum distance between ground and camera in meters (Perspective Camera only). Default is 250.

maxDistance : number

Maximum distance between ground and camera in meters (Perspective Camera only). Default is ellipsoid radius * 8.

minZoom : number

How far you can zoom in, in meters (Orthographic Camera only). Default is 0.

maxZoom : number

How far you can zoom out, in meters (Orthographic Camera only). Default is Infinity.

keyPanSpeed : number

Number of pixels moved per push on array key. Default is 7.

minPolarAngle : number

Minimum vertical orbit angle (in degrees). Default is 0.5.

maxPolarAngle : number

Maximum vertical orbit angle (in degrees). Default is 86.

minAzimuthAngle : number

Minimum horizontal orbit angle (in degrees). If modified, should be in [-180,0]. Default is -Infinity.

maxAzimuthAngle : number

Maximum horizontal orbit angle (in degrees). If modified, should be in [0,180]. Default is Infinity.

handleCollision : boolean

Handle collision between camera and ground or not, i.e. whether you can zoom underground or not. Default is true.

enableDamping : boolean

Enable damping or not (simulates the lag that a real camera operator introduces while operating a heavy physical camera). Default is true.

dampingMoveFactor : boolean

the damping move factor. Default is 0.25.

lookAtCoordinate ( params: CameraUtils~CameraTransformOptions|Extent, isAnimated: boolean ) : Promise

Changes the center of the scene on screen to the specified in lat, lon. See Coordinates for conversion. This function allows to change the central position, the zoom, the range, the scale and the camera orientation at the same time. The zoom has to be between the [getMinZoom(), getMaxZoom()]. Zoom parameter is ignored if range is set The tilt's interval is between 4 and 89.5 degree

Returns

A promise that resolves when transformation is complete

Parameters

  • params —

    camera transformation to apply
    • params.zoom (optional) —

      zoom
    • params.scale (optional) —

      scale
  • isAnimated —

    Indicates if animated

pan ( pVector: vector ) : Promise

Displaces the central point to a specific amount of pixels from its current position. The view flies to the desired coordinate, i.e.is not teleported instantly. Note : The results can be strange in some cases, if ever possible, when e.g.the camera looks horizontally or if the displaced center would not pick the ground once displaced.

Camera

Wrapper around Three.js camera to expose some geographic helpers.

Constructor

Camera ( crs: string, width: number, height: number, options: object )

Parameters

  • crs —

    The camera's coordinate projection system.
  • width —

    The width (in pixels) of the view the camera is associated to.
  • height —

    The height (in pixels) of the view the camera is associated to.
  • options (optional) —

    Options for the camera.
    • options.cameraThree (optional) —

      A custom Three.js camera object to wrap around.
    • options.type (default: CAMERA_TYPE.PERSPECTIVE) —

      The type of the camera. See CAMERA_TYPE.

Properties

crs : string

The camera's coordinate projection system.

camera3D : object

The Three.js camera that is wrapped around.

width : number

The width of the camera.

height : number

The height of the camera.

_preSSE : number

The precomputed constant part of the screen space error.

Methods

adjustAltitudeToAvoidCollisionWithLayer ( view: View, elevationLayer: ElevationLayer, minDistanceCollision: number )

Test for collision between camera and a geometry layer (DTM/DSM) to adjust camera position. It could be modified later to handle an array of geometry layers. TODO Improve Coordinates class to handle altitude for any coordinate system (even projected one)

Parameters

  • view —

    The view where we test the collision between geometry layers and the camera
  • elevationLayer —

    The elevation layer (DTM/DSM) used to test the collision with the camera. Could be another geometry layer.
  • minDistanceCollision —

    The minimum distance allowed between the camera and the surface.

position ( crs: string ) : Coordinates

Return the position in the requested CRS, or in camera's CRS if undefined.

Returns

Coordinates object holding camera's position.

Parameters

  • crs (optional) —

    If defined (e.g 'EPSG:4326'), the camera position will be returned in this CRS.

resize ( width: number, height: number )

Resize the camera to a given width and height

Parameters

  • width —

    The width to resize the camera to.
  • height —

    The height to resize the camera to.

setPosition ( position: Coordinates )

Set the position of the camera using a Coordinates object. If you want to modify the position directly using x,y,z values then use camera.camera3D.position.set(x, y, z)

Parameters

  • position —

    The new position of the camera.

Types

CAMERA_TYPE : object

  • PERSPECTIVE: number —

    Perspective type of camera
  • ORTHOGRAPHIC: number —

    Orthographic type of camera

Stores the different types of camera usable in iTowns.

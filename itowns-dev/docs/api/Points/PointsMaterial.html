PointsMaterial

Constructor

PointsMaterial ( options: object )

Parameters

  • options (default: {}) —

    The options
    • options.size (optional) —

      size point
    • options.mode (default: MODE.COLOR) —

      display mode.
    • options.overlayColor (default: new THREE.Vector4(0, 0, 0, 0)) —

      overlay color.
    • options.intensityRange (default: new THREE.Vector2(0, 1)) —

      intensity range.
    • options.applyOpacityClassication (optional) —

      apply opacity classification on all display mode.
    • options.classification (optional) —

      define points classification.

Example

// change color category classification
const pointMaterial = new PointsMaterial();
pointMaterial.classification[3].color.setStyle('red');
pointMaterial.recomputeClassification();

Properties

classification : Classification

points classification.

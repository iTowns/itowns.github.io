Fetcher

Utilitary to fetch resources from a server using the fetch API.

Static Methods

json ( url: string, options: Object ) : Promise.<Object>

Little wrapper over fetch to get some JSON.

Returns

Promise containing the JSON object.

Parameters

  • url —

    The URL of the resources to fetch.
  • options —

    Fetch options (passed directly to fetch()), see the syntax for more information.

multiple ( baseUrl: string, extensions: Object, options: Object ) : Array.<Promise>

Wrapper over fetch to get a bunch of files sharing the same name, but different extensions.

Returns

An array of promises, containing all the files, organized by their extensions (see the example below).

Parameters

  • baseUrl —

    The shared URL of the resources to fetch.
  • extensions —

    An object containing arrays. The keys of each of this array are available fetch type, such as text, json or even arrayBuffer. The arrays contains the extensions to append after the baseUrl (see example below).
  • options —

    Fetch options (passed directly to fetch()), see the syntax for more information.

Example

itowns.Fetcher.multiple('http://geo.server/shapefile', {
    // will fetch:
    // - http://geo.server/shapefile.shp
    // - http://geo.server/shapefile.dbf
    // - http://geo.server/shapefile.shx
    // - http://geo.server/shapefile.prj
    arrayBuffer: ['shp', 'dbf', 'shx'],
    text: ['prj'],
}).then(function _(result) {
    // result looks like:
    result = {
        shp: ArrayBuffer
        dbf: ArrayBuffer
        shx: ArrayBuffer
        prj: string
    };
});

texture ( url: string, options: Object ) : Promise.<THREE.Texture>

Wrapper around THREE.TextureLoader.

Returns

Promise containing the THREE.Texture.

Parameters

  • url —

    The URL of the resources to fetch.
  • options —

    Fetch options (passed directly to fetch()), see the syntax for more information. Note that THREE.js docs mentions withCredentials, but it is not actually used in THREE.TextureLoader.

Atmosphere extends GeometryLayer

It's layer to simulate Globe atmosphere. There's 2 modes : simple and realistic (atmospheric-scattering).

The atmospheric-scattering it is taken from :

  • Atmosphere Shader From Space (Atmospheric scattering)
  • Accurate Atmospheric Scattering (NVIDIA GPU Gems 2).

Constructor

Atmosphere ( id: string, options: Object )

Parameters

  • id (default: atmosphere) —

    The id of the layer Atmosphere.
  • options (optional) —

    options layer.
    • options.Kr (optional) —

      Kr is the rayleigh scattering constant.
    • options.Km (optional) —

      Km is the Mie scattering constant.
    • options.ESun (optional) —

      ESun is the brightness of the sun.
    • options.g (optional) —

      constant g that affects the symmetry of the scattering.
    • options.innerRadius (optional) —

      The inner (planetary) radius
    • options.outerRadius (optional) —

      The outer (Atmosphere) radius
    • options.wavelength (optional) —

      The constant is the wavelength (or color) of light.
    • options.scaleDepth (optional) —

      The scale depth (i.e. the altitude at which the atmosphere's average density is found).
    • options.mieScaleDepth (optional) —

      not used.

Methods

attach ( layer: Layer )

Attach another layer to this one. Layers attached to a GeometryLayer will be available in geometryLayer.attachedLayers.

Parameters

  • layer —

    The layer to attach, that must have an update method.

defineLayerProperty ( propertyName: string, defaultValue: *, onChange: function )

Defines a property for this layer, with a default value and a callback executed when the property changes.

When changing the property, it also emits an event, named following this convention: ${propertyName}-property-changed, with ${propertyName} being replaced by the name of the property. For example, if the added property name is frozen, it will emit a frozen-property-changed.

Parameters

  • propertyName —

    The name of the property, also used in the emitted event name.
  • defaultValue —

    The default set value.
  • onChange (optional) —

    The function executed when the property is changed. Parameters are the layer the property is defined on, and the name of the property.

Example

The emitted event has some properties assigned to it 
event = {
    new: {
        ${propertyName}: * // the new value of the property
    },
    previous: {
        ${propertyName}: * // the previous value of the property
    },
    target: Layer // the layer it has been dispatched from
    type: string // the name of the emitted event
}

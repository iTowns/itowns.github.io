C3DTBoundingVolume

bounding volume Used to represent bounding volumes and viewer request volumes. The input bounding volume (from the dataset) can be a box, a sphere or a region. Regions are transformed to spheres internally for simplification of parsing and to speed up computations such as culling.

Constructor

C3DTBoundingVolume ( )

Properties

type : C3DTilesTypes

Used by 3D Tiles extensions (e.g. C3DTBatchTableHierarchyExtension) to know in which context (i.e. for which 3D Tiles class) the parsing of the extension should be done.

initialVolumeType : String

the initial volume type to be able to dissociate spheres and regions if needed since both are converted to spheres (one of C3DTilesBoundingVolumeTypes)

volume : THREE.Box3|THREE.Sphere

The 3D bounding volume created. Can be a THREE.Box3 for bounding volumes of types box or a THREE.Sphere for bounding volumes of type sphere or region.

extensions : object

3D Tiles extensions of the bounding volume stored in the following format: {extensioName1: extensionObject1, extensioName2: extensionObject2, ...}

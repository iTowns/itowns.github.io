C3DTileset

A 3D Tiles Tileset.

Constructor

C3DTileset ( )

Properties

type : C3DTilesTypes

Used by 3D Tiles extensions (e.g. C3DTBatchTableHierarchyExtension) to know in which context (i.e. for which 3D Tiles class) the parsing of the extension should be done.

asset : object

Generic information about the tileset, see [asset specification]https://github.com/CesiumGS/3d-tiles/blob/master/specification/schema/asset.schema.json

properties : object

Properties associated with the tileset, see tileset specification

geometricError : number

see tileset specification

extensionsUsed : Array.<string>

see tileset specification

extensionsRequired : array

see tileset specification

tiles : Array.<object>

an array holding all the tiles (not their content which is stored in the attribute object3d of the layer). This list represents a flattened tileset.

extensions : object

Extensions of the tileset in the form: {extensioName1: extensionObject1, extensioName2: extensionObject2, ...}

Methods

parseTiles ( tile: object, baseURL: string, parent: object, registeredExtensions: object )

Recursion on the 3DTiles tileset (which is a tree) to parse the tiles (nodes of the tree).

Parameters

  • tile —

    current tile
  • baseURL —

    url of the source tileset and tiles
  • parent —

    parent tile (used for computing the transform matrix from local to global coordinates)
  • registeredExtensions —

    3D Tiles extensions registered in the C3DTilesLayer (see C3DTExtensions

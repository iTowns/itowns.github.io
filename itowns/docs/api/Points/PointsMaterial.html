PointsMaterial

Constructor

PointsMaterial ( options: object )

Parameters

  • options (default: {}) —

    The options
    • options.size (default: 1) —

      point size
    • options.mode (default: PNTS_MODE.COLOR) —

      display mode.
    • options.shape (default: PNTS_SHAPE.CIRCLE) —

      rendered points shape.
    • options.overlayColor (default: new THREE.Vector4(0, 0, 0, 0)) —

      overlay color.
    • options.classificationScheme (optional) —

      LUT for point classification colorization.
    • options.discreteScheme (optional) —

      LUT for other discret point values colorization.
    • options.gradient (optional) —

      Descrition of the gradient to use for continuous point values. (Default value will be the 'SPECTRAL' gradient from Utils/Gradients)
    • options.sizeMode (default: PNTS_SIZE_MODE.VALUE) —

      point cloud size mode. Only 'VALUE' or 'ATTENUATED' are possible. VALUE use constant size, ATTENUATED compute size depending on distance from point to camera.
    • options.minAttenuatedSize (default: 3) —

      minimum scale used by 'ATTENUATED' size mode
    • options.maxAttenuatedSize (default: 10) —

      maximum scale used by 'ATTENUATED' size mode

Example

// change color category classification
const pointMaterial = new PointsMaterial();
pointMaterial.classification[3].color.setStyle('red');
pointMaterial.recomputeClassification();

Properties

options.intensityRange : THREE.Vector2

intensity range (default value will be [1, 65536] if not defined at Layer level).

options.elevationRange : THREE.Vector2

elevation range (default value will be [0, 1000] if not defined at Layer level).

options.angleRange : THREE.Vector2

scan angle range (default value will be [-90, 90] if not defined at Layer level).

classificationScheme : Scheme

Color scheme for point classification values.

discreteScheme : Scheme

Color scheme for all other discrete values.

gradients : object

Descriptions of all available gradients.

gradient : object

Description of the gradient to use for display.

gradientTexture : THREE.CanvasTexture

The texture generate from the choosen gradient.

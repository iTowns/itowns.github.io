OGC3DTilesIonSource extends Source

An object defining the source connection to a 3DTiles asset of a Cesium ion server.

Constructor

OGC3DTilesIonSource ( source: Object )

Parameters

  • source —

    An object that can contain all properties of an OGC3DTilesIonSource and Source. Only accessToken and assetId are mandatory.
    • source.accessToken —

      The Cesium ion access token used to retrieve the resource.
    • source.assetId —

      The id of the asset on Cesium ion.

Properties

isOGC3DTilesIonSource : boolean

Used to check if this source is an OGC3DTilesIonSource. Set to true. You should not change this, as it is used internally for optimisation.

accessToken : string

The Cesium ion access token used to retrieve the resource.

assetId : string

The id of the asset on Cesium ion.

Minimap

A widget for minimap

Constructor

Minimap ( view: GlobeView, layer: ColorLayer, options: Object )

Parameters

  • view —

    The iTowns view the minimap should be linked to. Only GlobeView is supported at the moment.
  • layer —

    The ColorLayer that should be displayed on the minimap.
  • options (optional) —

    The minimap optional configuration.
    • options.parentElement (default: view.domElement) —

      The parent HTML container of the div which contains minimap widgets.
    • options.size (optional) —

      The size of the minimap. It is a number that describes both width and height in pixels of the minimap.
    • options.width (default: 150) —

      The width in pixels of the minimap.
    • options.height (default: 150) —

      The height in pixels of the minimap.
    • options.position (default: 'bottom-left') —

      Defines which position within the parentElement the minimap should be displayed to. Possible values are top, bottom, left, right, top-left, top-right, bottom-left and bottom-right. If the input value does not match one of these, it will be defaulted to bottom-left.
    • options.translate (optional) —

      An optional translation of the minimap.
      • options.translate.x (optional) —

        The minimap translation along the page x-axis.
      • options.translate.y (optional) —

        The minimap translation along the page y-axis.
    • options.cursor (optional) —

      An html element or an HTML string describing a cursor showing minimap view camera target position at the center of the minimap.
    • options.minScale (default: 1/2000) —

      The minimal scale the minimap can reach.
    • options.maxScale (default: 1/1_250_000) —

      The maximal scale the minimap can reach.
    • options.zoomRatio (default: 1/30) —

      The ratio between minimap camera zoom and view camera zoom.
    • options.pitch (default: 0.28) —

      The screen pixel pitch, used to compute view and minimap scale.

Properties

domElement : HTMLElement

An html div containing the minimap.

parentElement : HTMLElement

The parent HTML container of this.domElement.

Style

A Style is a class that defines the visual appearance of FeatureCollection and Feature. It is taken into account when drawing them in textures that will be placed onto tiles.

As there are five basic elements present in Features, there are also five main components in a Style object:

  • fill is for all fillings and polygons
  • stroke is for all lines and polygons edges
  • point is for all points
  • text contains all Label related things
  • icon defines the appearance of icons attached to label.

Many style property can be set to functions. When that is the case, the function's return type must necessarily be the same as the types (other than function) of the property. For instance, if the fill.pattern property is set to a function, this function must return an Image, a Canvas, or String. The first parameter of functions used to set Style properties is always an object containing the properties of the features displayed with the current Style instance.

Constructor

Style ( params: StyleOptions )

Parameters

  • params (default: {}) —

    An object that contain any properties (order, zoom, fill, stroke, point, text or/and icon) and sub properties of a Style (@see StyleOptions).

Example

const style = new itowns.Style({
     stroke: { color: 'red' },
     point: { color: 'white', line: 'red' },
});

const source = new itowns.FileSource(...);

const layer = new itowns.ColorLayer('foo', {
     source: source,
     style: style,
});

view.addLayer(layer);

Properties

order : Number

Order of the features that will be associated to the style. It can helps sorting and prioritizing features if needed.

fill : Object

Polygons and fillings style.

fill.color : String|function|THREE.Color

Defines the main color of the filling. Can be any valid color string. Default is no value, indicating that no filling needs to be done. If the Layer is a GeometryLayer you can use THREE.Color.

fill.pattern : Image|Canvas|String|Object|function

Defines a pattern to fill the surface with. It can be an Image to use directly, an url to fetch the pattern or an object containing the url of the image to fetch and the transformation to apply. from. See [this example] (http://www.itowns-project.org/itowns/examples/#source_file_geojson_raster) for how to use.

fill.pattern.source : Image|String

The image or the url to fetch the pattern image

fill.pattern.cropValues : Object

The x, y, width and height (in pixel) of the sub image to use.

fill.pattern.color : THREE.Color

Can be any [valid color string]

fill.opacity : Number|function

The opacity of the color or of the pattern. Can be between 0.0 and 1.0. Default is 1.0. For a GeometryLayer, this opacity property isn't used.

fill.base_altitude : Number|function

Only for GeometryLayer, defines altitude for each coordinate. If base_altitude is undefined, the original altitude is kept, and if it doesn't exist then the altitude value is set to 0.

fill.extrusion_height : Number|function

Only for GeometryLayer and if user sets it. If defined, polygons will be extruded by the specified amount.

stroke : Object

Lines and polygons edges.

stroke.color : String|function|THREE.Color

The color of the line. Can be any valid color string. Default is no value, indicating that no stroke needs to be done. If the Layer is a GeometryLayer you can use THREE.Color.

stroke.opacity : Number|function

The opacity of the line. Can be between 0.0 and 1.0. Default is 1.0. For a GeometryLayer, this opacity property isn't used.

stroke.width : Number|function

The width of the line. Default is 1.0.

stroke.base_altitude : Number|function

Only for GeometryLayer, defines altitude for each coordinate. If base_altitude is undefined, the original altitude is kept, and if it doesn't exist then the altitude value is set to 0.

point : Object

Point style.

point.color : String|function

The color of the point. Can be any valid color string. Default is no value, indicating that no point will be shown.

point.radius : Number|function

The radius of the point, in pixel. Default is 2.0.

point.line : String|function

The color of the border of the point. Can be any valid color string. Not supported for GeometryLayer.

point.width : Number|function

The width of the border, in pixel. Default is 0.0 (no border).

point.opacity : Number|function

The opacity of the point. Can be between 0.0 and 1.0. Default is 1.0. Not supported for GeometryLayer.

point.base_altitude : Number|function

Only for GeometryLayer, defines altitude for each coordinate. If base_altitude is undefined, the original altitude is kept, and if it doesn't exist then the altitude value is set to 0.

point.model : Object

3D model to instantiate at each point position

text : Object

All things Label related.

text.field : String|function

A string representing a property key of a FeatureGeometry enclosed in brackets, that will be replaced by the value of the property for each geometry. For example, if each geometry contains a name property, text.field can be set to {name}. Default is no value, indicating that no text will be displayed.

It's also possible to create more complex expressions. For example, you can combine text that will always be displayed (e.g. foo) and variable properties (e.g. {bar}) like the following: foo {bar}. You can also use multiple variables in one field. Let's say for instance that you have two properties latin name and local name of a place, you can write something like {name_latin} - {name_local} which can result in Marrakesh - مراكش for example.

text.color : String|function

The color of the text. Can be any valid color string. Default is #000000.

text.anchor : String|Array.<Number>|function

The anchor of the text relative to its position (see Label for the position). Can be one of the following values: top, left, bottom, right, center, top-left, top-right, bottom-left or bottom-right. Default is center.

It can also be defined as an Array of two numbers. Each number defines an offset (in fraction of the label width and height) between the label position and the top-left corner of the text. The first value is the horizontal offset, and the second is the vertical offset. For example, [-0.5, -0.5] will be equivalent to center.

text.offset : Array|function

The offset of the text, depending on its anchor, in pixels. First value is from left, second is from top. Default is [0, 0].

text.padding : Number|function

The padding outside the text, in pixels. Default is 2.

text.size : Number|function

The size of the font, in pixels. Default is 16.

text.wrap : Number|function

The maximum width, in pixels, before the text is wrapped, because the string is too long. Default is 10.

text.spacing : Number|function

The spacing between the letters, in em. Default is 0.

text.transform : String|function

A value corresponding to the CSS property text-transform. Default is none.

text.justify : String|function

A value corresponding to the CSS property text-align. Default is center.

text.opacity : Number|function

The opacity of the text. Can be between 0.0 and 1.0. Default is 1.0.

text.font : Array|function

A list (as an array of string) of font family names, prioritized in the order it is set. Default is Open Sans Regular, Arial Unicode MS Regular, sans-serif.

text.haloColor : String|function

The color of the halo. Can be any valid color string. Default is #000000.

text.haloWidth : Number|function

The width of the halo, in pixels. Default is 0.

text.haloBlur : Number|function

The blur value of the halo, in pixels. Default is 0.

icon : Object

Defines the appearance of icons attached to label.

icon.source : String

The url of the icons' image file.

icon.id : String

The id of the icons' sub-image in a vector tile data set.

icon.cropValues : String

the x, y, width and height (in pixel) of the sub image to use.

icon.anchor : String

The anchor of the icon compared to the label position. Can be left, bottom, right, center, top-left, top-right, bottom-left or bottom-right. Default is center.

icon.size : Number

If the icon's image is passed with icon.source and/or icon.id, its size when displayed on screen is multiplied by icon.size. Default is 1.

icon.color : String|function

The color of the icon. Can be any valid color string. It will change the color of the white pixels of the icon source image.

icon.opacity : Number|function

The opacity of the icon. Can be between 0.0 and 1.0. Default is 1.0.

Methods

applyToCanvasPolygon ( txtrCtx: CanvasRenderingContext2D, polygon: Path2D, invCtxScale: Number, canBeFilled: Boolean )

Applies the style.fill to a polygon of the texture canvas.

Parameters

  • txtrCtx —

    The Context 2D of the texture canvas.
  • polygon —

    The current texture canvas polygon.
  • invCtxScale —

    The ratio to scale line width and radius circle.
  • canBeFilled —

    true if feature.type == FEATURE_TYPES.POLYGON.

applyToHTML ( domElement: Element ) : undefined|Promise.<HTMLImageElement>

Applies this style to a DOM element. Limited to the text and icon properties of this style.

Returns

for a text label: undefined. for an icon: a Promise resolving with the HTMLImageElement containing the image.

Parameters

  • domElement —

    The element to set the style to.

getTextAnchorPosition ( ) : Array.<Number>

Gets the values corresponding to the anchor of the text. It is proportions, to use with a translate() and a transform property.

Returns

Two percentage values, for x and y respectively.

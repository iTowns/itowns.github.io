StyleContext

StyleContext stores metadata of one FeatureGeometry that are needed for its style computation: type of feature and what is needed (fill, stroke or draw a point, etc.) as well as where to get its properties and its coordinates (for base_altitude).

Constructor

StyleContext ( )

Properties

zoom : number

Current zoom to display the FeatureGeometry.

collection : Object

The FeatureCollection to which the FeatureGeometry is attached.

properties : Object

Properties of the FeatureGeometry.

type : string

Geometry type of the feature. Can be point, line, or polygon.

featureStyle : StyleOptions|function

StyleOptions object (or a function returning one) to get style information at feature and FeatureGeometry level from the data parsed.

coordinates : Coordinates

The coordinates (in world space) of the last vertex (x, y, z) set with setLocalCoordinatesFromArray(). private properties:

worldCoord : Coordinates

@private Coordinates object to store coordinates in world space.

localCoordinates : Coordinates

@private Coordinates object to store coordinates in local space.

worldCoordsComputed : boolean

@private Have the world coordinates already been computed from the local coordinates?

feature : Feature

@private The itowns feature of interest.

geometry : FeatureGeometry

@private The FeatureGeometry to compute the style.

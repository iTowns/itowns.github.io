FeatureGeometry

Constructor

FeatureGeometry ( feature: Feature )

Parameters

  • feature —

    geometry

Properties

extent : Extent

The 2D extent containing all the points composing the geometry.

indices : Array.<Object>

Contains the indices that define the geometry. Objects stored in this array have two properties, an offset and a count. The offset is related to the overall number of vertices in the Feature.

properties : Object

Properties of the geometry. It can be anything specified in the GeoJSON under the properties property.

Methods

closeSubGeometry ( count: number, feature: Feature )

After you have pushed new the coordinates of sub geometry without startSubGeometry, this function close sub geometry. The sub geometry stored in indices, see constructor for more information.

Parameters

  • count —

    count of vertices
  • feature —

    the feature containing the geometry

pushCoordinates ( feature: Feature, coordIn: Coordinates )

Push new coordinates in vertices buffer.

Parameters

  • feature —

    the feature containing the geometry
  • coordIn —

    The coordinates to push.

pushCoordinatesValues ( feature: Feature, coordIn: Object, coordProj: Coordinates )

Push new values coordinates in vertices buffer without any transformation. No geographical conversion is made or the normal doesn't stored.

Parameters

  • feature —

    the feature containing the geometry
  • coordIn —

    An object containing the coordinates values to push.
    • coordIn.x —

      the x coordinate (in a local system).
    • coordIn.y —

      the y coordinate (in a local system).
    • coordIn.normal (optional) —

      the normal on coordinates (only for EPSG:4978 projection).
  • coordProj (optional) —

    An optional argument containing the geodesic coordinates in EPSG:4326 It allows the user to get access to the feature coordinates to set style.base_altitude.

startSubGeometry ( count: number, feature: Feature )

Add a new marker to indicate the starting of sub geometry and extends the vertices buffer. Then you have to push new the coordinates of sub geometry. The sub geometry stored in indices, see constructor for more information.

Parameters

  • count —

    count of vertices
  • feature —

    the feature containing the geometry

updateExtent ( )

update geometry extent with the last sub geometry extent.

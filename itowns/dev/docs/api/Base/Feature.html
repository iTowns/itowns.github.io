Feature

This class improves and simplifies the construction and conversion of geographic data structures. It's an intermediary structure between geomatic formats and THREE objects.

Warning, the data (extent or Coordinates) can be stored in a local system. To use vertices or extent in Feature.crs projection, it's necessary to transform Coordinates or Extent by FeatureCollection.matrixWorld.

// To have feature extent in featureCollection.crs projection:
feature.extent.applyMatrix4(featureCollection.matrixWorld);

// To have feature vertex in feature.crs projection:
coord.crs = feature.crs;
coord.setFromArray(feature.vertices)
coord.applyMatrix4(featureCollection.matrixWorld);

Constructor

Feature ( type: string, collection: FeatureCollection )

Parameters

  • type —

    type of Feature. It can be 'point', 'line' or 'polygon'.
  • collection —

    Parent feature collection.

Properties

type : string

Geometry type, can be point, line, or polygon.

vertices : Array.<number>

All the vertices of the Feature.

normals : Array.<number>

All the normals of the Feature.

size : number

the number of values of the array that should be associated with a coordinates. The size is 3 with altitude and 2 without altitude.

hasRawElevationData : boolean

indicates if the geographic coordinates, from original source, has an elevation, the coordinates has a third coordinate.

crs : string

Geographic or Geocentric coordinates system.

geometries : Array.<FeatureGeometry>

An array containing all FeatureGeometry.

extent : Extent

The extent containing all the geometries composing the feature.

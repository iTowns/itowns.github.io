Classification

Every lidar point can have a classification assigned to it that defines the type of object that has reflected the laser pulse. Lidar points can be classified into a number of categories including bare earth or ground, top of canopy, and water. The different classes are defined using numeric integer codes in the files.

Properties

visible : boolean

category visibility,

name : string

category name,

color : THREE.Color

category color,

opacity : number

category opacity,

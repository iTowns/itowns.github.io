Static Methods

parse ( buffer: ArrayBuffer, options: Object ) : Promise

Parse b3dm buffer and extract THREE.Scene and batch table

Returns

  • a promise that resolves with an object containig a THREE.Scene (gltf) and a batch table (batchTable).

Parameters

  • buffer —

    the b3dm buffer.
  • options —

    additional properties.
    • options.gltfUpAxis (default: 'Y') —

      embedded glTF model up axis.
    • options.urlBase —

      the base url of the b3dm file (used to fetch textures for the embedded glTF model).
    • options.doNotPatchMaterial (optional) —

      disable patching material with logarithmic depth buffer support.
    • options.opacity (default: 1.0) —

      the b3dm opacity.
    • options.frustumCulled (optional) —

      enable frustum culling.
    • options.overrideMaterials (optional) —

      override b3dm's embedded glTF materials. If true, a threejs MeshBasicMaterial is set up. config.overrideMaterials can also be a threejs Material in which case it will be the material used to override.

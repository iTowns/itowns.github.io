VectorTileParser

Static Methods

parse ( file: ArrayBuffer, options: Object ) : Promise

Parse a vector tile file and return a Feature or an array of Features. While multiple formats of vector tile are available, the only one supported for the moment is the Mapbox Vector Tile.

Returns

A Promise resolving with a Feature or an array a Features.

Parameters

  • file —

    The vector tile file to parse.
  • options —

    Options controlling the parsing ParsingOptions.
    • options.in —

      Object containing all styles, layers and informations data, see InformationsData.
      • options.in.styles —

        Object containing subobject with informations on a specific style layer. Styles available is by layer.id and by zoom.
      • options.in.layers —

        Object containing subobject with
    • options.out —

      options indicates how the features should be built, see FeatureBuildingOptions.

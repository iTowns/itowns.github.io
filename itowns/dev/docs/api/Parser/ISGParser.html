ISGParser

The ISGParser module provides a parse method. This method takes the content of a ISG file in, and returns a GeoidGrid. the GeoidGrid contains all the necessary attributes and methods to access the ISG data in iTowns.

Static Methods

parse ( isg: string, options: Object ) : Promise.<GeoidGrid>

Parses an ISG file content and returns a corresponding GeoidGrid.

Returns

A promise resolving with a GeoidGrid, which contains all the necessary attributes and methods to access ISG file data.

Parameters

  • isg —

    The content of the ISG file to parse.
  • options —

    An object gathering the optional parameters to pass to the parser.
    • options.in (default: {}) —

      Information on the ISG data.
      • options.in.crs (default: 'EPSG:4326') —

        The Coordinates Reference System (CRS) of the ISG data. It must be a geographic CRS, and must be given as an EPSG code.

VpcLayer extends PointCloudLayer

A layer for Virtual Point Clouds (VPC) datasets. See PointCloudLayer class for documentation on base properties.

Constructor

VpcLayer ( id: string, config: Object )

Parameters

  • id —

    Unique id of the layer.
  • config —

    See PointCloudLayer for base pointcloud options.

Example

// Create a new VPC layer
const vpcSource = new VpcSource({
    url: 'https://data.geopf.fr/chunk/telechargement/download/lidarhd_fxx_ept/vpc/index.vpc',
});

const vpcLayer = new VpcLayer('VPC', {
    source: vpcSource,
});

View.prototype.addLayer.call(view, vpcLayer);

Properties

isVpcLayer : boolean

Used to checkout whether this layer is a VpcLayer. Default is true. You should not change this, as it is used internally for optimisation.

Atmosphere extends GeometryLayer

It's layer to simulate Globe atmosphere. There's 2 modes : simple and realistic (atmospheric-scattering).

The atmospheric-scattering it is taken from :

  • Atmosphere Shader From Space (Atmospheric scattering)
  • Accurate Atmospheric Scattering (NVIDIA GPU Gems 2).

Constructor

Atmosphere ( id: string, options: Object )

Parameters

  • id (default: atmosphere) —

    The id of the layer Atmosphere.
  • options (optional) —

    options layer.
    • options.Kr (optional) —

      Kr is the rayleigh scattering constant.
    • options.Km (optional) —

      Km is the Mie scattering constant.
    • options.ESun (optional) —

      ESun is the brightness of the sun.
    • options.g (optional) —

      constant g that affects the symmetry of the scattering.
    • options.innerRadius (optional) —

      The inner (planetary) radius
    • options.outerRadius (optional) —

      The outer (Atmosphere) radius
    • options.wavelength (optional) —

      The constant is the wavelength (or color) of light.
    • options.scaleDepth (optional) —

      The scale depth (i.e. the altitude at which the atmosphere's average density is found).
    • options.mieScaleDepth (optional) —

      not used.

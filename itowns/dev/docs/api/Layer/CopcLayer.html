CopcLayer extends PointCloudLayer

A layer for Cloud Optimised Point Cloud (COPC) datasets. See PointCloudLayer class for documentation on base properties.

Constructor

CopcLayer ( id: string, config: Object )

Parameters

  • id —

    Unique id of the layer.
  • config —

    See PointCloudLayer for base pointcloud options.

Example

// Create a new COPC layer
const copcSource = new CopcSource({
    url: 'https://s3.amazonaws.com/hobu-lidar/autzen-classified.copc.laz',
    crs: 'EPSG:4978',
    colorDepth: 16, // bit-depth of 'color' attribute (either 8 or 16 bits)
});

const copcLayer = new CopcLayer('COPC', {
    source: copcSource,
});

View.prototype.addLayer.call(view, copcLayer);

loadData ( elt: PointCloudNode, context: Object, layer: PointCloudLayer, distanceToCamera: THREE.Box3 ) : Array.<pointCloudNode>

Load the data of a node. Send a promise to get the data (if not already sent) and add the result to the node when resolve. Check the visiblility of children to see if the need to be updated as well.

Returns

The child nodes to update (if needed).

Parameters

  • elt —

    The element (node) to load data.
  • context —

    The context.
  • layer —

    The layer on wich the node is attach.
  • distanceToCamera —

    The distance between the camera and the node.

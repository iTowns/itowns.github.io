GeoidLayer

Creates a new instance of GeoidLayer.

Constructor

GeoidLayer ( id: string, config: Object )

Parameters

  • id —

    An unique identifier for the layer.
  • config —

    The layer configuration. All elements in it will be merged as is in the layer. For example, if the configuration contains three elements name, protocol, extent, these elements will be available using layer.name or something else depending on the property name. Only config.source parameter is mandatory.
    • config.source —

      The source of the geoid data displayed by the GeoidLayer. It is mandatory that the source data for a GeoidLayer be parsed into a GeoidGrid. You can refer to GTXParser, GDFParser and ISGParser to see how three standard geoid height grid file formats are parsed into GeoidGrid.

Example

// Create a GeoidLayer from a GTX geoid heights file.
const geoidLayer = new GeoidLayer('geoid', {
    source: new FileSource({
        url: 'url-to-some-GTX-geoid-heights-file.gtx',
        crs: 'EPSG:4326',
        format: 'application/gtx',
    }),
});

COGSource extends Source

An object defining the source of resources to get from a COG file. It inherits from Source.

Constructor

COGSource ( source: Object )

Parameters

  • source —

    An object that can contain all properties of a COGSource and Source. Only url is mandatory.

Example

// Create the source
const cogSource = new itowns.COGSource({
    url: 'https://cdn.jsdelivr.net/gh/iTowns/iTowns2-sample-data/cog/orvault.tif',
});

// Create the layer
const colorLayer = new itowns.ColorLayer('COG', {
    source: cogSource,
});

// Add the layer
view.addLayer(colorLayer);

Properties

zoom : Object

Object containing the minimum and maximum values of the level, to zoom in the source.

zoom.min : number

The minimum level of the source. Default value is 0.

zoom.max : number

The maximum level of the source. Default value is Infinity.

url : string

The URL of the COG.

pool : GeoTIFF.Pool

Pool use to decode GeoTiff.

defaultAlpha : number

Alpha byte value used if no alpha is present in COG. Default value is 255.

tileWidth : number

Tile width in pixels. Default value use 'geotiff.getTileWidth()'.

tileHeight : number

Tile height in pixels. Default value use 'geotiff.getTileHeight()'.

resampleMethod : number

The desired resampling method. Default is 'nearest'.

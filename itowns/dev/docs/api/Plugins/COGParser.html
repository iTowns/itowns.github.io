COGParser

The COGParser module provides a parse method that takes a COG in and gives a THREE.DataTexture that can be displayed in the view.

It needs the geotiff library to parse the COG.

Example

GeoTIFF.fromUrl('http://image.tif')
    .then(COGParser.parse)
    .then(function _(texture) {
        var source = new itowns.FileSource({ features: texture });
        var layer = new itowns.ColorLayer('cog', { source });
        view.addLayer(layer);
    });

Static Methods

parse ( data: Object, options: Object ) : Promise.<THREE.DataTexture>

Parse a COG file and return a THREE.DataTexture.

Returns

A promise resolving with a THREE.DataTexture.

Parameters

    Data passed with the Tile extent

  • options —

    Options (contains source)
    • options.in

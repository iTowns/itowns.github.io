OrientedImageSource extends Source

OrientedImageSource is a specific source used to load oriented images.

Constructor

OrientedImageSource ( source: Object )

Parameters

  • source —

    Configuration object
    • source.url —

      Url for all the textures.
    • source.orientationsUrl —

      Json Url, using GeoJSon format to represent points, it's a set of panoramic position and orientation.
    • source.calibrationUrl —

      Json url, representing a set of camera. see CameraCalibrationParser This Url must contains {sensorId} and {cameraId}, and these pattern will be replaced to build the Url, to find the good texture for each camera for each panoramic.

VpcSource extends Source

An object defining the source of Entwine Point Tile data. It fetches and parses the main configuration file of Entwine Point Tile format, ept.json.

Constructor

VpcSource ( config: Object )

Parameters

  • config —

    The configuration, see Source for available values.
    • config.colorDepth (optional) —

      Color depth (in bits). Either 8 or 16 bits. By defaults it will be set to 8 bits for LAS 1.2 and 16 bits for later versions (as mandatory by the specification).

Properties

isEntwinePointTileSource : boolean

Used to checkout whether this source is a EntwinePointTileSource. Default is true. You should not change this, as it is used internally for optimisation.

url : string

The URL of the directory containing the whole Entwine Point Tile structure.

sources : Array.<Object>|Array.<PointCloudSource>

Array of all the source described in the VPC. initialized with mockSource that will be replace by COPC or EPT Source as soon as any data need to be loaded.

Methods

extentInsideLimit ( extent: Extent ) : boolean

Tests if an extent is inside the source limits.

Returns

True if the extent is inside the limit, false otherwise.

Parameters

  • extent —

    Extent to test.

instantiate ( source: object )

We created several objects mocking a source at the intantiation of the vpc layer. The following method will be called when we need to instantiate the real source (COPC or EPT). The mock source contain the url to instantiate the source as well as a promise that will be resolve when the source.whenReady promise will resolve. To avoid instanciating several time the same source, we use a cache to store the whenReady promise of the source (cachedSources).

Parameters

  • source —

    The mock source used to instantiate the real source.
    • source.url —

      The url of the source to instanciate.

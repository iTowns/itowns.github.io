OGC3DTilesSource extends Source

An object defining the source connection to a 3DTiles dataset from a web server.

Constructor

OGC3DTilesSource ( source: Object )

Parameters

  • source —

    An object that can contain all properties of OGC3DTilesSource and of Source. Only url is mandatory.
    • source.url —

      The URL of the tileset json.

Properties

isOGC3DTilesSource : boolean

Used to check if this source is an isOGC3DTilesSource. Set to true. You should not change this, as it is used internally for optimisation.

url : string

The URL of the tileset json.

Methods

extentInsideLimit ( extent: Extent ) : boolean

Tests if an extent is inside the source limits.

Returns

True if the extent is inside the limit, false otherwise.

Parameters

  • extent —

    Extent to test.

Generates an url from an extent. This url is a link to fetch the resources inside the extent.

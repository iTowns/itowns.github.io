# iTowns public web site

This site is powered by [Jekyll][jekyllSite].

To serve it localy, you should first head to [Jekyll doc][jekyllDoc] to get
started, but the TLDR is:

1. You need to have [Ruby][rubySite] and [RubyGems][rubygemsSite] installed.
2. Clone this repo and cd into it.
3. Run the following command to install dependencies:
    ```bash
    bundle install
    ```
4. Run the following command to serve iTowns website locally:
    ```bash
    bundle exec jekyll serve
    ```

[jekyllSite]: https://jekyllrb.com/ "Jekyll website (home page)"
[jekyllDoc]: https://jekyllrb.com/docs/ "Jekyll quickstart page"
